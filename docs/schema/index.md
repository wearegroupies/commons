# Schema Overview

The Groupies Commons schema uses a two-layer model: **canonical facts** and **enrichment context**.

Canonical facts are the fixed, verifiable core of a record — event date, venue, headliner, status. Enrichment context adds structured but softer detail: set lists, support acts, media links, and contributor notes. The two layers are kept separate so the integrity of the canonical record is never compromised by uncertain or crowd-sourced additions.
